Grutils Documentation

randomElem

This function returns a random element from the array given.

Usage

import { randomElem } from "https://deno.land/x/grutils@0.0.5/mod.ts";

console.log(randomElem([1, 2, 3, 4, 5]));

inArr

Returns true if the element given is part of the array given, otherwise it returns false.

Usage

import { inArr } from "https://deno.land/x/grutils@0.0.5/mod.ts";

console.log(inArr([1, 2, 3, 4], 4));

trimEnds

Returns the given string with the number of characters given trimmed on each side.

Usage

import { trimEnds } from "https://deno.land/x/grutils@0.0.5/mod.ts";

console.log(trimEnds("!Hello!")); // Hello
console.log(trimEnds("!!Hello!!", 2)); // Hello

Notes

  • The number of characters given is 1 by default.

indexOfEnd

Returns the last index of the substring given of the string given, Behaves the same as String.prototype.indexOf() for other details.

If the substring is of length 1, it returns the same as String.prototype.indexOf() and therefore you should use that.

Usage

import { indexOfEnd } from "https://deno.land/x/grutils@0.0.5/mod.ts";

console.log(indexOfEnd("hello there!", "ere")) // prints 11
